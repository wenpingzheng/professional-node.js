#DOS命令行

##1. dir  == directory

```angular2html

 dir
 
```
s：查看当前目录已经其所有子目录的文件和文件夹。 

/a：查看包括隐含文件的所有文件。 

/ah：只显示出隐含文件。
 

/w：以紧凑方式（一行显示5个文件）显示文件和文件夹。
 
/p：以分页方式（显示一页之后会自动暂停）显示。
 

##2. cd  = change directory


cd 目录名：进入特定的目录。如果看到有个目录显示为：abcdef ghi 就
 
输入：cdabcdef.ghi进入该目录。
 
cd\ 退回到根目录。
 
cd..退回到上一级目录。
